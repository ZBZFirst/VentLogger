# VentLogger v1.5

First GitHub distribution build.

## Highlights

- Local-first Android charting workflow.
- Respiratory group hierarchy loaded from the app asset.
- Add/Edit patient template setup.
- Log screen creates new timestamped entries.
- Review screen supports sorting, popup summaries, edit mode, and confirmed delete.
- Comment category for free-form text.
- Photo and video gallery attachments per log entry.
- Settings page with theme selection and bulk CSV export.

## APK

Attach the debug APK to the GitHub Release:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Notes

This is a `v1.x` GitHub-hosted distribution build. The project will continue
through `v1.9`; after that, the next major release should roll over to `v2.0`.
